# sync

Run the full project sync pipeline.

Arguments: $ARGUMENTS

This skill delegates to the platform-specific command files:
1. **Claude Code**: `.claude/commands/sync.md` handles the full sync pipeline
2. **Gemini CLI**: `.gemini/commands/sync.md` handles the full sync pipeline

## Pipeline Steps

1. Write a daily session log entry to `memory/YYYY-MM-DD.md`
2. Update `memory/MEMORY.md` index via `bun scripts/sync-md.ts`
3. Auto-add entry to `CHANGELOG.md [Unreleased]` if missing
4. Run the documentation audit — aborts if audit fails
5. Guard against committing sensitive files
6. Create a PR branch (if on main), commit, push, and open a GitHub PR

If audit fails, fix the reported issue before re-running `/sync`.

## PR Language Rule

All PR titles and bodies generated by this command **must be written in English**, regardless of the active session language. Never write Korean or any other language in `gh pr create` or `gh pr edit` calls.
